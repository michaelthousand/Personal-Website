1000 Workouts (better name pending)

3/19/23: version 1.0 is now live. Click here to be redirected to the app.

This is a web application for tracking user's workouts to help them keep up with their fitness goals. User inputs are stored in a binary database that is linked to their username. This allows them to view their wworkout history and track some basic statistics. More functionality will be added in the future.

Version 1.0 includes individual profiles that allow users to see some basic statistics.

It also allows users to view their individual workouts, and edit or delete incorrect entries.

Upcoming Updates:

  • More statistics
  • Workout planning
  • Customizable user profiles
  • Mobile optimization
